    Welcome to ProcureChain Documentation¶

    Overview¶

    ProcureChain is a comprehensive, enterprise-grade application designed to streamline and automate the entire procurement lifecycle. It provides a unified platform for internal procurement staff and external vendors, enhancing efficiency, transparency, and strategic decision-making through advanced AI-powered analytics.

    Key Features¶

    üè¢ Internal Procurement Application¶

    • Dashboard: Central hub with real-time metrics on sourcing events, vendors, contracts, and requisitions
    • Sourcing & Tendering: Manage RFQs, REOIs, and Tenders from approved requisitions
    • Vendor Management: Onboard vendors, track performance and risk scores, manage compliance
    • Contract Lifecycle Management: Centralized repository with status tracking and expiration alerts
    • Procurement Workflow: Full requisition-to-order process with multi-stage approval
    • Financial Oversight: Monitor purchase orders and review/approve vendor invoices
    • User Management: Role-based access control (RBAC) system
    • AI-Powered Analytics Hub:
      • Spend Analysis with trend identification
      • Predictive Forecasting using historical data
      • What-If Scenario modeling
      • Conversational Insights with natural language queries
    • Vendor Support Desk: Complete helpdesk for vendor support tickets

    üåê Vendor Portal¶

    • Self-Service Onboarding: Simple registration process
    • Dashboard: View open opportunities and performance metrics
    • Electronic Bidding: Secure bid submission and tracking
    • Compliance Center: Document upload and management
    • Financials: Track purchase orders and submit invoices
    • Contract Management: View awarded contracts
    • Support Center: Create and track support tickets

    Technology Stack¶

    ProcureChain is built with modern, enterprise-grade technologies:

    • Framework: Next.js (App Router)
    • Language: TypeScript
    • Styling: Tailwind CSS
    • UI Components: ShadCN/UI
    • Database: PostgreSQL (Neon Serverless)
    • ORM: Drizzle ORM
    • Authentication: Firebase Authentication
    • AI: Genkit with Google's Gemini models
    • Deployment: Docker / Firebase App Hosting
