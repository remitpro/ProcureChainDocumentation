    First Steps¶

    Congratulations on installing ProcureChain! This guide will walk you through your first actions and help you get familiar with the system.

    Initial Login¶

    Staff Portal Login¶

    1. Navigate to http://localhost:9002/login (or your configured URL)
    2. Enter your super admin credentials:
    3. Email: The email you set in SUPER_ADMIN_EMAIL
    4. Password: The password you set in SUPER_ADMIN_PASSWORD
    5. Click Sign In

    You'll be redirected to the staff dashboard.

    Vendor Portal Login¶

    1. Navigate to http://localhost:9002/vendor/login
    2. Use a seeded vendor account:
    3. Email: vendor1@example.com
    4. Password: password123
    5. Click Sign In

    You'll be redirected to the vendor dashboard.

    Exploring the Staff Portal¶

    Dashboard Overview¶

    The dashboard provides at-a-glance metrics:

    • Active Sourcing Events: Current RFQs, REOIs, and Tenders
    • Vendor Count: Total registered vendors
    • Expiring Contracts: Contracts nearing expiration
    • Pending Requisitions: Requisitions awaiting approval

    Creating Your First User¶

    1. Navigate to Users in the sidebar
    2. Click Add User
    3. Fill in the form:
    4. Name
    5. Email
    6. Role (Admin, Manager, or Staff)
    7. Click Create User

    The user will be created in both the database and Firebase Authentication.

    User Roles

    • Admin: Full system access
    • Manager: Can approve requisitions and manage vendors
    • Staff: Can create requisitions and view data

    Adding a Vendor¶

    1. Navigate to Vendors in the sidebar
    2. Click Add Vendor
    3. Fill in vendor details:
    4. Name
    5. Email
    6. Category
    7. Contact information
    8. Click Create Vendor

    The vendor account will be created and they'll receive login credentials.

    Creating a Requisition¶

    1. Navigate to Requisitions
    2. Click New Requisition
    3. Fill in the form:
    4. Title
    5. Description
    6. Category
    7. Estimated value
    8. Justification
    9. Click Submit

    The requisition enters the approval workflow.

    Creating an RFQ¶

    1. Navigate to Sourcing RFQs
    2. Click New RFQ
    3. Select an approved requisition
    4. Set:
    5. Title
    6. Description
    7. Closing date
    8. Evaluation criteria
    9. Invite vendors
    10. Click Publish

    Invited vendors can now submit bids.

    Evaluating Bids¶

    1. Navigate to Sourcing RFQs
    2. Click on an RFQ with submitted bids
    3. View all bids in the Bids tab
    4. Score each bid against evaluation criteria
    5. Click Award on the winning bid

    Creating a Contract¶

    1. Navigate to Contracts
    2. Click New Contract
    3. Select:
    4. Contract template
    5. Awarded requisition/RFQ
    6. Review the generated contract
    7. Click Create

    The contract is created and the vendor can view it in their portal.

    Exploring the Vendor Portal¶

    Dashboard Overview¶

    Vendors see:

    • Open Opportunities: Active RFQs and Tenders
    • My Bids: Submitted quotations
    • Performance Score: Vendor rating
    • Compliance Status: Document verification status

    Submitting a Bid¶

    1. Navigate to Opportunities
    2. Click on an open RFQ
    3. Click Submit Bid
    4. Fill in:
    5. Quoted amount
    6. Delivery timeline
    7. Additional notes
    8. Upload supporting documents
    9. Click Submit

    Your bid is now visible to procurement staff.

    Uploading Compliance Documents¶

    1. Navigate to Compliance
    2. Click on a required document
    3. Click Upload
    4. Select your file
    5. Click Submit

    Documents are reviewed by procurement staff.

    Viewing Contracts¶

    1. Navigate to Contracts
    2. View all awarded contracts
    3. Click on a contract to view details
    4. Download PDF if needed

    Creating a Support Ticket¶

    1. Navigate to Support
    2. Click New Ticket
    3. Fill in:
    4. Subject
    5. Category
    6. Description
    7. Click Submit

    Procurement staff will respond to your ticket.

    Exploring AI Analytics¶

    Spend Analysis¶

    1. Navigate to Analytics Spend Analysis
    2. View spend breakdown by:
    3. Category
    4. Vendor
    5. Time period
    6. Identify trends and savings opportunities

    Predictive Forecast¶

    1. Navigate to Analytics Forecast
    2. Select a category
    3. View predicted spend for upcoming periods
    4. Adjust parameters to refine forecast

    What-If Scenarios¶

    1. Navigate to Analytics Scenarios
    2. Create a new scenario:
    3. "What if we reduce IT spend by 15%?"
    4. View the projected impact
    5. Compare multiple scenarios

    Conversational Insights¶

    1. Navigate to Analytics Insights
    2. Ask questions in natural language:
    3. "Which vendors have the highest performance scores?"
    4. "Show me contracts expiring in the next 30 days"
    5. "What's our total spend on office supplies this year?"
    6. Get AI-generated answers based on real-time data

    Customizing Your Experience¶

    User Profile¶

    1. Click your avatar in the top right
    2. Select Profile
    3. Update:
    4. Name
    5. Email
    6. Password
    7. Notification preferences

    Notification Settings¶

    1. Navigate to Settings Notifications
    2. Configure:
    3. Email notifications
    4. In-app alerts
    5. Frequency preferences

    Theme Settings¶

    1. Click the theme toggle in the header
    2. Switch between light and dark mode

    Managing Contract Templates¶

    Creating a Template¶

    1. Navigate to Settings Contract Templates
    2. Click New Template
    3. Enter:
    4. Template name
    5. Template content (using Handlebars syntax)
    6. Available variables:
    7. {{vendorName}}
    8. {{contractValue}}
    9. {{startDate}}
    10. {{endDate}}
    11. And more...
    12. Click Save

    Using a Template¶

    When creating a contract, select your template and it will be populated with data from the requisition and vendor.

    Setting Up Evaluation Criteria¶

    1. Navigate to Settings Evaluation Criteria
    2. Click Add Criterion
    3. Enter:
    4. Criterion name (e.g., "Price", "Quality", "Delivery Time")
    5. Weight (percentage)
    6. Click Save

    These criteria are used when scoring bids.

    Best Practices¶

    For Procurement Staff¶

    1. Regular Reviews: Check the dashboard daily for pending approvals
    2. Vendor Management: Keep vendor information up to date
    3. Contract Monitoring: Set up alerts for expiring contracts
    4. Documentation: Maintain detailed requisition justifications
    5. Fair Evaluation: Use consistent criteria for all bids

    For Vendors¶

    1. Complete Profile: Fill in all company information
    2. Compliance: Keep all documents current
    3. Timely Bids: Submit quotations before deadlines
    4. Accurate Pricing: Provide detailed, accurate quotes
    5. Responsive: Reply to support tickets promptly
